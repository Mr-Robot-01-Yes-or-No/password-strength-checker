# 🔐 Password Strength Checker

## 📌 Overview

This project is a Python-based password strength checker designed to evaluate how secure a password is against common cyber attacks such as brute-force and guessing attacks.

It analyzes multiple security factors and provides real-time feedback to help users create stronger and more secure passwords.

---

## 🚀 Features

* Evaluates password strength (Weak / Medium / Strong)
* Checks for:

  * Minimum length
  * Uppercase and lowercase letters
  * Numbers
  * Special characters
* Provides suggestions to improve weak passwords
* Runs continuously until a strong password is entered
* Clean and user-friendly terminal output

## ▶️ How to Run

1. Clone this repository
2. Run the script:
   python password_checker.py
3. Enter a password and follow the suggestions
